# 🌿 Student Health & Wellness Portal

A modern full-stack web application designed to help students maintain their overall well-being by providing easy access to health and wellness resources.

This platform offers mental health support materials, fitness programs, nutrition guidance, and general wellness resources through an intuitive dashboard interface.

---

## 📌 Project Overview

The **Student Health & Wellness Portal** is a role-based system that allows:

- **Students** to access wellness resources and enroll in programs
- **Admins** to manage content, programs, and analytics

The application focuses on promoting healthier lifestyles, reducing stress, and improving student wellness.
